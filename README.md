## **Clonar el repositorio**

### Paso 1:
```shell
git clone https://github.com/SrAlberto-eng/AutoStock.git
```

### Paso 2 - Ubicarte en el entorno del proyecto:
```shell
cd AutoStock
```

### Paso 3 - Crear entonro virtual:
```shell
python -m venv myenv
```

### Paso 4 - Activar entorno virtual:
```shell
# Windows Power Shell
.\venv\Scrips\Activate.ps1

# Windows Command Prompt
.\venv\Scripts\activate.bat

# macOS/Linux
.\venv\Scripts\activate
```

### Paso 4.1 - Opcional y si lo requiere, instalar la última versión de pip:
```shell
python -m pip install --upgrade pip
```

### Paso 5 - Instalar requirements.txt (herramientas y bibliotecas necesarias para el proyecto)
```shell
pip install requirements.txt
```

## **Para crear un feature**

> Debemos nombrar el branch con "feature/nombre_del_feature" para identificar en que se está trabajando.

### Crear un branch:
```shell
git checkout -b feature/nombre_del_feature
```

### Trabajar y hacer commits:
```shell
git add .
git commit -m "feat: ejemplo de comit de un feature"
```

### Subir el branch:
```shell
git push origin feature/nombre_del_feature
```
